Culture4Kids - Crowdfunding Backend

The crowdfunding backend project was built using Python and Django Rest Framework, implementing a robust, scalable architecture to support Aboriginal Community Controlled Organisations in creating programs for children in care. The project also included advanced functionality like filtering, searching, and inline pledge management via the Django admin panel. Skills demonstrated include API design, database modeling, user authentication, and deploying backend services on platforms like Heroku.
